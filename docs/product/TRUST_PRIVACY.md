# Trust & Privacy Considerations

## 1. User Control

- Users decide what data to share and with whom
- Easy account/data deletion

## 2. Data Security

- End-to-end encryption (in transit)
- Sensitive data encrypted at rest
- No sensitive data in notifications

## 3. Transparency

- Open source codebase
- Clear privacy policy
- All actions logged (with user consent)

## 4. Emergency Handling

- SOS works offline, caches and retries
- No data shared without explicit consent

## 5. Community Trust

- Inclusive, respectful community standards
- Code of Conduct enforced

---

> _Trust is the foundation of safety. SOMA is built to earn and keep it._
