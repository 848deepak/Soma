# User Stories

## As a Woman User

- I want to send an SOS alert with my location in one tap, so I can get help quickly.
- I want to link my partner or trusted contact, so they’re notified in emergencies.
- I want to log my daily symptoms and mood, so I can track my well-being.
- I want to track my cycle, so I can better understand my health.
- I want to see insights and trends, so I can make informed decisions.
- I want my data to be private and secure, so I feel safe using the app.

## As a Partner/Trusted Contact

- I want to receive instant alerts if my partner is in danger, so I can respond quickly.
- I want to view my partner’s safety status, so I know they’re okay.

## As an Admin/NGO

- I want to analyze anonymized safety data, so I can identify trends and improve community safety.

## Edge Cases

- As a user, I want the app to work in low or no network conditions, so I’m not left unprotected.
- As a user, I want to trigger SOS even if the app is in the background or locked.

## Trust & Privacy

- As a user, I want full control over my data and sharing preferences.
