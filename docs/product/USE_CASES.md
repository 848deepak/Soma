# Use Cases

## 1. Emergency SOS

**Actor:** Woman User
**Scenario:** User taps SOS. App shares location and alert with linked partner(s). Optionally, notifies local authorities.

## 2. Partner Linking

**Actor:** Woman User, Partner
**Scenario:** User invites partner. Partner accepts. Both can view each other’s safety status.

## 3. Daily Check-in

**Actor:** Woman User
**Scenario:** User logs mood, symptoms, and notes. Data is visualized in insights dashboard.

## 4. Cycle Tracking

**Actor:** Woman User
**Scenario:** User enters cycle data. App predicts next cycle, provides health tips.

## 5. Insights Dashboard

**Actor:** Woman User
**Scenario:** User views trends, receives personalized safety and health insights.

## 6. Privacy Control

**Actor:** Woman User
**Scenario:** User manages data sharing, can delete account/data at any time.

## 7. Low Network/Emergency

**Actor:** Woman User
**Scenario:** App caches SOS and logs, sends when network is restored.

## 8. Admin/NGO Analytics

**Actor:** Admin/NGO
**Scenario:** Admin views anonymized data to identify safety trends.
