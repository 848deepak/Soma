# Contributing to SOMA – Women Safety App

Thank you for your interest in contributing! Your support helps us build a safer world for women.

## How to Contribute

1. **Fork the repo** and create your branch from `main`
2. **Write clear, well-tested code** (TypeScript, React Native)
3. **Add/Update documentation** as needed
4. **Open a pull request** with a clear description
5. **Participate in code reviews**

## Code Style

- Follow [eslint](eslint.config.js) and [prettier] rules
- Use descriptive commit messages
- Write/maintain tests (Jest, Detox)

## Reporting Issues

- Use [GitHub Issues](https://github.com/YOUR_ORG/womenproject/issues)
- Provide steps to reproduce, expected/actual behavior

## Community

- Be respectful and inclusive
- See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

## Getting Started

- See [README.md](README.md) for setup
- For major changes, open an issue first to discuss
